    Social Media Web App
    • Reactivities is a social media web application featuring user login & registration, profile viewing & editing, image uploading, real-time chatting, infinite scrolling, user following, and a persistent integrated PostgreSQL database. An active deployment of Reactivities is accessible from its GitHub repository.

    GitHub Repo
  • Synesthesia
    Music Platformer Game
    • Synesthesia is an abstract cat platformer featuring a dynamic audio system, syncing music with the player's movement in real-time. Synesthesia's visuals are a mix of traditional pixel art and PS1-inspired 3D backgrounds. Builds for HTML5, Windows, & macOS are available for free on Itch.io.
    • Made With: Godot, GDScript, Photoshop, & Illustrator
    Play on Itch.io
  • Meedee
    MIDI Platformer Game
    • Meedee is a procedurally generated 2D platforming game. Levels are generated from a user-provided MIDI file, which the game uses to construct terrain based on the MIDI's contents. The MIDI file's music is played in time with the player's movements as they progress through the procedurally-generated level.
    • Made With: Python
    GitHub Repo
  • Thisorthat
    Trivia Maze Game
    • Thisorthat is a trivia maze game developed collaboratively by a small team at University of Washington for a Software Development class. The goal of the game is to escape the labyrinthine halls of Ikea by successfully answering trivia questions. Thisorthat was made adhering to the Scrum project management framework.
    • Made With: Java
